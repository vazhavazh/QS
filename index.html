Getting started 🔽

About I-Deviator

I-Deviator is a web-based solution for navigational equipment on ships.
You can learn more about I-Deviator by visiting www.i-deviator.cz

Watch our informational video for an in-depth understanding of how I-Deviator evolves Magnetic Compass Adjustment.

How to launch

Launch your browser and navigate to https://ideviator.com

Ensure your browser is updated to the latest version (If logging fails in the initially selected default browser, try access in using Microsoft Edge browser for better compatibility).

If you encounter any issues accessing the site:
  1. Contact your satellite internet provider and request them to add the domains *.ideviator.com and *.i-deviator.cz to their trusted list.
  2. Inform about the problem to our technical support at support@i-deviator.cz.

Instant translation

Default I-Deviator software language is English. We strongly recommend using default language to ensure the accurate completion of the procedure with the highest level of precision. If you experience any difficulties understanding the instructions provided in English, you may utilise the translation feature to convert the guidance into your native language. Please note that while this function is available for convenience, it is advisable to refer to the original English version for the most reliable results.

To use this translation feature, we also recommend using Google Chrome.

A stable internet connection is also required, preferably with a speed of at least 1 megabit per second. On slower connections, applying the translation may take some time, potentially up to several minutes. If the translation is not applied after an extended period or if you encounter visual issues related to slow internet speeds, try refreshing the page in your browser.

For instant translation

1. While using the I-Deviator, right-click with your mouse on a blank field to display the dropdown menu.

2. Choose option - translate to (your default language). On the right upper corner will appear menu with default language.

3. If you need to translate into another language, click on three dots – choose option Choose another language

4. In appeared language list choose language to which you want to translate text.

5. Click button Translate.

Management Board 🚢

The Management Board is an administrative section designed for company managers to monitor the status of a fleet using the I-Deviator solution. This section contains:

  • Review the history of Magnetic Compass Adjustment procedures,
  • Access electronic Compass Error Log Book records for each fleet vessel in one place.

The Management Board analyzes data to:

  • Identify problematic areas in the parameters of the magnetic compass and gyrocompass,
  • Indicate of non-compliance record-keeping of the Compass Error Log Book by the crew,
  • Calculate the validity period of a vessel's Deviation Card and notify about its upcoming expiration.

Upon successful login to your account, you will be automatically redirected to the Management Board. If you navigate to another section or need to return to the main screen of the Management Board, you can do so by clicking the corresponding button on the left side of the screen.

Main Screen of the Management Board

The main screen serves as the starting point for monitoring the fleet using the I-Deviator solution. From here, you can navigate to other parts of the Management Board as well as internal screens containing data entered by vessel crews in their accounts. This approach maximizes access to all available information without requiring additional logins or risking unintended changes using a vessel's account.

Magnetic Compass 🧭 Adjustment

General information

The magnetic compass adjustment procedure is conducted in three stages:

  1. The information gathering stage: ship's position and particulars, model of magnetic compass, details of compensating tools, other relevant information.
  2. The deviation identification stage. During this stage the ship performs swinging according to corresponding instructions and provides deviation data of the magnetic compass at present condition.If residual deviation within 3° in vessels of 500 or more gross tonnage and, 4° in vessels below 500 gross tonnage, then no additional adjustment is needed (according to ISO 25862:2019(E)).
  3. The adjustment stage. At this stage vessel's staff is provided with guidance for second maneuvering and movement the correcting tools for adjustment the compass. Upon successful completion of the deviation adjustment procedure, an electronic document: "Deviation Card" is issued, which can be downloaded to your device and printed.

Adjustment procedure divided in to sub-stages with progress indication above the chat window.

Required data

At the initial stage of the procedure, I-Deviator will collect information about the vessel and the compass.

You can prepare the required materials in advance:

  1. Photos of the compass binnacle in its current condition, taken from a distance of 2 to 3 meters, showing both the forward and aft sections, as well as a photo of the compass bowl taken from a distance of about 0.5 meters.
  2. A copy of the most recent deviation table issued by a shore-based adjuster.
  3. Additionally, if available, the most recent compass maintenance service report.
  4. A copy of the last 3 pages of the compass error logbook (if using a paper logbook) or a file with the last month's records (if using an electronic logbook).
  5. 1-3 photos showing the current position of the compensating magnets or correcting knobs.

If you are performing the procedure for the first time, instructions on how to access the correcting tools will be provided by I-Deviator during the procedure.

In addition to the materials, you may also gather the following information in advance:

  1. The name of the manufacturer, the model, and the serial number of the compass installed on the vessel.

    2. This information should be taken from the label on the compass bowl. For example: Tokyo Keiki SH-165, 23321.
    You will be asked to provide these details separately during the procedure.

  2. Are there any bubbles inside the compass bowl?
  3. Does the compass card move freely and smoothly without getting stuck?
  4. If there are spare compensating magnets on board.
  5. If there is a spare compass bowl on board.

After completing the maneuvers and adjustments, you will need to upload screenshots of the maneuver from the ECDIS and photos of the new positions of the correcting tools.

This section will display all the previous adjustment procedures, both completed and those in progress. Using this section, you can always return to the process of chosen procedure or download the 'Deviation card' after its completion.

You can always return to the 'Magnetic Compass Adjustment' section by pressing the corresponding button in the sidebar.

Some messages will include forms for entering data, interactive elements such as lists , switches and upload file buttons.

Enter the information carefully and double-check the entered data before sending - you will not have the opportunity to edit the response. If you made mistake, the request process must be started from the beginning.

You can always scroll to a specific message and browse the information you entered, or responses from I-Deviator or an expert. Sometimes responses will contain interactive elements or forms for data entry. Fill in the data elements of such questions before sending a response.
  3. The action area at the bottom. Contains interactive action elements, such as:

Magnetic anomaly zone

At the beginning of the deviation adjustment procedure, I-Deviator will ask confirmation that the ship is not located in Magnetic Anomaly Zone (map will be presented). It will not recommend conducting maneuvers there and will be better to carry out swinging after the ship has left such area. Meantime, you can commence the procedure by starting with the information gathering stage.

Vessel and compass information

The first stage of the procedure is dedicated to collecting general information about the vessel, the compass installed on it, and other information required to obtain the 'Deviation Card'.

Carefully enter the data and fill out the forms. If data is entered incorrectly, it may be necessary to start the procedure again.

  1. When entering text data, use the field in the lower action area to send the response by clicking the message Send button or pressing the Enter key.
  2. If there are interactive elements in the message or action area, select the necessary options and then proceed to send the message button. Where it is possible I-Deviator will highlight the element with a red outline if incorrect or mandatory information is missing.

    In some cases, the send button will be inactive until you enter the required information or select an option in the interactive document, for example, when entering the position of the ship and choosing a direction.

    The main part of the entered data is used in the formation of the Deviation Card and will be displayed unchanged in it. Therefore, enter values using the provided examples, without adding extra information.

  3. Upload file size is limited to 5MB. The previous deviation table can be uploaded as an image or a digital document.
  4. Select the compass manufacturer using the interactive list element. If your compass manufacturer is not on the list, invite an expert by clicking 'None of the options fits'.
  5. Download and print the compass instructions provided by I-Deviator, it will guide you thru necessary corrective action. .

Correctors

To obtain information about the current position of the correctors, I-Deviator will provide forms with interactive elements corresponding to the compass installed on the ship.
Using the compass instructions, determine and indicate the presence, location, or absence of correctors.

1. B and C compensation magnets.
The form is divided into three scales:
- Fore-and-aft Port Side magnets;
- Fore-and-aft Starboard Side magnets;
- Athwartship magnets;

You can enter the required number of magnets, specify their position on the scale and the direction the red pole of the magnet is facing.
Use the buttons to add and remove magnets on the scale.

You can submit an empty form if compensating magnets are absent.

Carefully enter the values, as an input error will lead to incorrect results for the entire procedure. If you encounter difficulties at this stage, invite an expert by clicking the corresponding button in the lower action area of the chat.


If a Tokyo Keiki Knob type (Tokimec) compass is installed on the ship, in the form for B and C correctors, indicate the value and color of the division scale.

2. D correctors.
Depending on the type of compass installed on the ship, you will be provided with a form to specify the location of correcting spheres or boxes with plates. Using the compass instructions, determine the values for these correctors and enter the data into the provided form.

Use the corrector presence toggle to enable or disable it if the sphere or plates in the box are absent.

3. Latitude correctors (Flinders bars).
Using the downloaded compass instructions, determine and input the data into the provided form for this type of correctors.

Manoeuvres

Before performing maneuvers, you will be provided with instructions.

All maneuvers should be carried out in full awareness of traffic safety and in favorable weather conditions which should permit to take compass readings with accuracy 0.2 – 0.3 degrees.

To ensure accurate magnetic compass adjustment, please keep all walkie-talkies, metal tools, and other metallic or electronic devices (Mobile phone, Tablet PC, etc) away from the compass during the adjustment procedure. The presence of these items can create magnetic interference, leading to incorrect readings and inaccurate adjustments.

If ORCA camera, STARLINK antenna or other electrical equipment is to be installed on compass deck, it shall be installed in accordance with IMO Resolution A.382(X) Magnetic compass carriage and performance standards - 8.4
The distance of the magnetic compass from electrical or magnetic equipment should be at least equal to the safe distance specified for the equipment and be to the satisfaction of the Administration.

After confirming the possibility to start maneuvers, I-Deviator will send a form for each course separately.

Enter values carefully, as an error at this stage may lead to incorrect results for the entire procedure.

1. Maneuvers based on the suggested Magnetic headings.
These maneuvers are performed first, and the deviation for each of the proposed courses will be calculated based on their results.

Set the vessel on the proposed Magnetic heading and after 2-3 minutes input the corresponding True heading in the appropriate box. Also, enter the variation in the respective field. The variation for each course can be entered separately.

In case deviation values on each of 4 cardinal and 4 inter-cardinal heading turned out to be less than +/-3.0 degrees, new Deviation Card will be issued without additional adjustment.

Instructions

If deviations above the norm are detected as a result of the maneuvers, I-Deviator will provide instructions for moving the correctors and details for carrying out additional maneuvers.

If you encounter difficulties at this stage, you can invite an expert.

In some cases, I-Deviator will automatically invite an expert.

Deviation card

Upon completion of the procedure, I-Deviator will offer to download and/or send the "Deviation card" digital document to the email specified in your profile.

If you face any difficulties in receiving the document or find an error in it, please contact support via email support@i-deviator.cz.

The Deviation card can be downloaded at any time from the "Request history" screen, using the corresponding download button.

Frequently Asked Questions

Q. How long standard Deviation card is valid?

A. Standard deviation card, including issued after adjustment, is valid within 2 years period. Although, some Flag Administrations or Shipowners may reduce such period to one year.

Q. What is maximum admissible value of residual deviation for vessel’s standard magnetic compass?

A. Maximum admissible value of deviation is +/- 3.0 degrees for vessels greater than 500 g.r.t.

Q. What coefficient B reflects?

A. Coefficient B reflects the value of deviation level on E and W directions: bigger coefficient – bigger value of deviation.

Q. What coefficient C reflects?

A. Coefficient C reflects the value of deviation level on N and S directions: bigger coefficient – bigger value of deviation.

Q. What coefficient D reflects?

A. Coefficient D reflects the value of deviation level on inter-cardinal directions: bigger coefficient – bigger value of deviation.

Q. How is coefficient B calibrating?

A. Coefficient B is calibrating by approaching or distancing fore-and-aft magnets to or from compass bowl.

Q. How is coefficient C calibrating?

A. Coefficient C is calibrating by approaching or distancing athwart-ship magnets to or from compass bowl.

Q. How is coefficient D calibrating?

A. Coefficient D is calibrating by approaching or distancing side spheres or boxes with soft iron plates to or from compass bowl.

Q. Spare magnetic compass. When and how spare magnetic compass should be adjusted?

A. Spare magnetic compass must be adjusted only after installation in binnacle on permanent basis by standard adjustment procedure.

Q. Why each measured deviation value may show "+" or "-" sign?
A. Among various reasons main are:
  • insufficient accuracy during take compass readings or keep walkie-talkie, mobile phone near the compass bowl
  • electrical or magnetic equipment close to the compass is recently added, removed or altered
  • vessel was recently involved in accident
  • on-board cargo has significant magnetic properties and located mostly on vessel’s port or starboard side.
Q. How many maneuvers are required for obtaining a compass deviation card?

One maneuver: during checking of compass deviation values on headings 000°, 045°, 090°, 135°, 180°, 225°, 270°, and 315°,
if the residual deviation on all these headings does not exceed the following limits:
• 4° for vessels below 500 gross tonnage;
• 3° for vessels of 500 gross tonnage or more according to ISO 25862:2019(E).

The second maneuver may be required depending on which headings residual deviation exceeds the above limits, combined with the repositioning of magnets.

Q. What factors determine the date specified in the certificate as the date of issuance?

A. The date of issue of the certificate is the date on which the first or, if applicable, the second adjustment maneuver was carried out.

Q. Why does the deviation obtained during the maneuvers differ from the one shown on the deviation card?

The deviation shown on the deviation card is calculated based on the coefficients listed on the card, as well as the formulas of Archibald Smith and Siméon Denis Poisson.

Highlighting of increased values

The cells of Gyro Error, Deviation and Correlation Value columns are highlighted with red background colour with different gradations in case of overestimated values in them.

Highlight the Gyro Error cell

In accordance with IMO Resolution A.424(XI) – Performance Standards for Gyro-Compasses (adopted in 1979), the required accuracy of a gyrocompass is defined as follows:
- under static conditions, the settle point error should not exceed ±0.75° × secant latitude, which corresponds to approximately ±0.75° to ±1.5° for latitudes between 0° and 60°.
- under dynamic conditions, the residual steady-state error must not exceed ±0.25° × secant latitude, or approximately ±0.25° to ±0.5° within the same latitude range.
However, gyrocompass manufacturers typically declare accuracy, usually between ±0.25° and ±0.5°, under normal operational conditions.

Taking both the regulatory requirements and current technological capabilities into account, Compass Error Log Book is programmed to monitor gyro error and notifies the crew when deviation may indicate misalignment or drift. If the gyro error exceeds 0.5°, the system cell will have a reddish background and indicates an elevated value. When the error falls between 0.5° and 1.0°, calibration is recommended - either by the crew, if the gyrocompass interface allows, or by a service provider. If the gyro error exceeds 1.0°, a formal calibration is required, as the deviation surpasses both the expected technological accuracy and the performance thresholds set by the IMO standard.

Determining the Gyro Error

Bridge Procedures Guide, Sixth Edition, 5.3.2 Gyro compass
The gyro usually supports several repeaters including a required repeater at the emergency steering position. Gyro repeaters on the bridge should be checked against the main gyro at least once per watch and after significant manoeuvring. Other repeaters should be checked frequently. Any anomalies should be corrected and the gyro repeaters resynchronised.

IMO Resolution A.424(XI) Performance standards for gyro compasses, 5.2.4
The maximum divergence in reading between the master compass and repeaters under all operational conditions should not exceed +/- 0.5°

It is recommended to check the synchronisation of the gyro repeaters with the master compass before determining the gyro error. And if the difference in readings will be more than 0,5 degrees, it is necessary to synchronise the repeater(s) according to the manufacturer's manual.
If the gyro error is significant (in line with IMO Resolution A.424(XI), Flag or Company requirements) and compass calibration is necessary, it must be performed in strict compliance with the manufacturer's manual. Calibration can be performed by the crew by making changes in the settings, using the gyro compass interface (if this is provided and allowed by the manufacturer) or by calling a service engineer on board.

Highlight the Deviation and the Correlation Value cells

According to ISO 25862:2019(E) A properly adjusted compass shall have a residual deviation within 3° in vessels of 500 or more gross tonnage and, 4° in vessels below 500 gross tonnage. Accuracies shall be within 2° for safe navigation.

In line with these requirements, Compass Error Log Book logbook is configured to indicate the crew if the Correlation value exceeds 2°, ensuring timely awareness and corrective action to maintain safe navigation.
The magnetic compass deviation values are highlighted starting from 3° and have wider gradation steps. A constantly overestimated deviation of more than 5° for several directions indicates that the magnetic compass requires adjustment.

Let's take a closer look at them:

Deviation Card Log Block

Deviation Card Log is used to record the received deviation cards of the vessel, as well as to calculate the Correlation Value field - the difference between the log and table deviation values for a certain course, calculated when adding an entry to the Compass Error Log Book. In addition, the data from the Deviation Card Log will help to notify in due time about deviation card expiry.

The Deviation Card Log can be filled in automatically or manually. If the magnetic compass adjustment procedure was performed with the I-Deviator, a deviation card record can be added automatically after the procedure is completed. This entry cannot be edited, but can be viewed by clicking on the edit icon in the entry bar.

If, when completing the magnetic compass adjustment procedure with the I-Deviator, a new deviation entry was not added automatically or the deviation card was issued by another company by the adjuster - a manual entry must be added to the log.

Adding a record to the logbook

To add a new entry to the Compass Error Logbook:

  1. Log in to the I-Deviator using your credentials.
  2. Use the Compass Error Logbook button on the left sidebar to navigate to the electronic logbook section.
  3. Click the Add record button below the log table.
  4. A form for adding an entry will appear.
  5. Choose one of the available entry modes:

    • Default – Used when weather conditions allow taking bearings.
    • Overcast - Used when weather conditions do not allow taking bearings.
    • Another Reason - Used when conditions other than weather prevent taking bearings.
    • Vessel in port – Used to indicate that the vessel is in port.

    Please note:
    Each entry mode has a Comment field where you can add a note about the current entry. For example, for a Vessel in port entry, you can specify the date range the vessel will be in port.

  6. Fill in the form fields.

    All fields except the Comment and Correlation Value fields are mandatory. For direction fields, select the appropriate value by clicking the radio button. Use the back ← and forward → keys on your keyboard to move between delimiters within a field. Enter degrees, time values using leading zeros, e.g., 001 instead of 1. Unfilled or incorrectly filled fields will be highlighted in red. To calculate the Correlation Value, data from the current deviation card must already be added to the Deviation Card Log on the Compass Maintenance tab.
  7. Select your role from the Observer list.

    Each role in the logbook should have a signature attached, which can be uploaded as an image file or signed on an electronic canvas. This is done once, and the logbook will use the signature when the associated role is selected.
    Please note:
    • When the crew changes, a new signature must be assigned to roles. This does not affect existing entries in the logbook.
    • A signature cannot be deleted, only replaced.
    • It is recommended to use the electronic canvas to create a signature.
  8. If necessary, add a comment to the entry.
    Please note:
    • The comment length is limited to 50 symbols.
    • For Overcast and Vessel in port modes, the Comment field includes an entry mode icon and placeholder text that can be changed.
  9. Save the entry by clicking the Save button

    10. After successfully adding the entry, you will be automatically returned to the log screen. If there are unfilled or incorrectly filled fields in the form, you will receive an error message, and the fields requiring attention will be highlighted in red for correction.

    Please note:
    • The saved entry will be placed in the log according to the date and time specified in the Date/UTC fields.
    • The entry can be edited within 7 days of its creation, after which it will be locked for editing.
    • The original entry and all its revisions will be saved in the platform's database but will not be explicitly displayed in the log. The edited entry will have a status indicating the change and the number of edits.
    • An entry can be backdated (up to 30 days from the current date). This option helps to fill in the log after a prolonged absence of an internet connection. The actual creation date of the entry will be saved separately to prevent constant untimely log keeping

Editing logbook record

To edit an existing log record, you need to:

  1. Move the cursor over the record Status cell
  2. If the record is not locked for editing, the status icon will change to a pencil.
  3. Click on the pencil icon to go to the record editing form.
  4. Make the required changes according to the rules of adding an record to the log and save the record by pressing the Save button.
Please note:
  • The saved record will be located in the logbook according to the specified date and time in the Date/UTC fields.
  • The record can be edited within 7 days from the date of its creation, after expiry of this period, the record will be blocked for editing.
  • The original record and all its revisions will be stored in the platform database, but will not be explicitly displayed in the logbook. The edited record will have a status indicating the change of the record and indicating the number of edits.

Searching and filtering data

To search through the logbook, use the search form above the logbook table:

  1. To search, type the query in the form and press Enter or click the search icon.
  2. To reset the search, delete the entered query text and press Enter or the search icon.

Please note:
The search is performed on data from all logbook fields except date.

To filter records by date or date range, use the function to select the displayed records by selected range:

  1. Click the button Select range of records by date to bring up a drop-down box with options for displaying logbook records filtered by date or date range.
  2. In the drop-down box, specify the date range, records for which you want to display. The first click on the date marks the start date, the second click on the end date. The selected range is highlighted.
    To select a specific date, double-click on it.
    The selected range is displayed below as text and can be reset with the Reset button.
  3. After specifying the range, click the Show records button. The drop-down box will close and the logbook table will display records included in the selected range.

To disable filtering of records by date range:

  1. Call the drop-down box again by clicking on the button Select range of records by date
  2. Set the Show logbook for all time switch to active. This will hide the range selection calendars.
  3. Click the Show records button. The drop-down box will disappear and the filtering of logbook records will be disabled.

Exporting data from the logbook to a PDF file

Logbook data can be exported to a PDF file prepared for printing or sending as an electronic file-document.

In order to export the logbook data to a PDF file:

  1. Click the button Download data to call the drop-down box with options for downloading the logbook records to a PDF file.
  2. In the drop-down box specify the date range, records for which you want to export to the file. The first click on the date marks the start date, the second click on the end date. The selected range is highlighted.
    If you want to export all logbook records, set the Download logbook for all time switch to active. This will hide the range selection calendars.
  3. Click the Download button. The prepared file will be downloaded according to the web browser settings.
Please note:
  • The file does not display the Status and Correlation Value columns, nor does it highlight cells with excessive values.
  • The file is as close as possible to the look of the classic paper Compass Error Log Book.
